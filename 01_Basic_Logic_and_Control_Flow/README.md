\## Problem 01: Find the Largest of Three Numbers



\*\*Input:\*\*  

Three numbers a, b, c



\*\*Output:\*\*  

The largest number among a, b, c



\*\*Algorithm:\*\*

1\. Read values a, b, and c

2\. If a >= b AND a >= c, print a

3\. Else if b >= a AND b >= c, print b

4\. Else print c
